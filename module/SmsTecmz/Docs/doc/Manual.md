## 申请接口信息

访问 [api.tecmz.com](https://api.tecmz.com) ，在用户中心 → 我的接口，购买开通「短信服务」接口。

![](https://ms-assets.modstart.com/data/image/2022/10/06/43372_oloe_8383.png)

分别申请短信签名和短信模板。

在我的接口查看 `AppId` 和 `AppSecret`。
